## 二、主题分组

### A. 新大众文艺的概念、历史与价值框架

D01—D05、D07、D12、D14—D17、D25、D28、D29。

主要处理“大众—人民”“代言—自言”“大众文化—新大众文艺”“人民性—大众化”等概念和历史关系。

### B. 生产装置、平台系统与媒介经验

D03、D06、D09、D13、D16、D17。

主要处理作者作为生产者、多模态创作、算法驱动、平台环境、流量代码、速度经验和媒介格式化。

### C. 日常生活审美化、消费文化与艺术边界

D10、D11、D18—D24。

主要处理生活审美化、审美生活化、文化研究与审美主义之争、消费符号、身体快感和生活美学。

### D. 审美经验、形式完成与艺术合法性

D08、D24、D26、D27。

主要处理创作者经验如何通过形式成为“一个经验”、接受者如何完成意义，以及作品如何在艺术场域中获得承认。

## 三、与当前论文选题最相关的精读顺序

建议按以下顺序构建理论链：

1. **D26 杜威《艺术即经验》**：建立“经历—经验—一个经验—艺术经验”的核心模型。
2. **D29 谢有顺《重新认识经验的力量》**：把握大众亲历经验与身体细节的材料优势。
3. **D15 刘大先《新大众文艺的认知转型与方法变革论纲》**：区分短暂体验与可传递经验。
4. **D12 黎杨全、谭歆跃《新大众文艺与自言现实主义的兴起》**：明确代言—自言、典型性—在场性的前沿论争。
5. **D28 洪治纲《论新大众文艺的创作形态及审美追求》**：定位“日常生活诗学”及其尚未展开的形式机制。
6. **D08 姚斯《审美经验与文学解释学》**：解释私人经验如何在接受中转化为公共经验。
7. **D06 本雅明《作为生产者的作者》**：考察媒介装置和生产关系对形式的规定。
8. **D27 布尔迪厄《艺术的法则》**：补足艺术合法性和评价场域。
9. **D09 晏青、万旭婷《幻影快感》**：反思数字速度对经验完整性的破坏。
10. **D16 徐婧《全息性社会系统》**：分析平台、算法与流量对经验的预先格式化。

## 四、综合研究结论

29份文献共同显示，现有研究已经较充分回答了以下问题：

- 大众为何能够成为文艺创作主体；
- 数字媒介如何降低表达与传播门槛；
- 日常生活为何成为新大众文艺的重要资源；
- 亲历经验为何具有在场感、细节密度和现实认识价值；
- 平台、算法和消费文化如何重构经验表达。

但尚未被充分整合的问题是：

> 亲历经验如何从生活材料转化为具有内在统一、情感节奏、媒介形式、公共可传达性和艺术合法性的艺术经验？

据此可以形成论文的核心判断：

> 经验真实只是新大众文艺的材料合法性和主体合法性，并非艺术性的充分条件。大众经验需要经过经验选择与整合、情感组织与距离化、媒介形式与节奏建构，并在接受和批评场域中获得公共意义，才能由私人亲历转化为艺术真实。
